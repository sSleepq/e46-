# 1. Теоретическая часть

## Подзапрос (Subquery)

**Подзапрос** — это SQL-запрос, который используется внутри другого запроса.

Он позволяет сначала получить промежуточный результат, а затем использовать его в основном запросе.

Подзапрос может находиться в:

* `WHERE`
* `FROM`
* `SELECT`

---

## Общий синтаксис

```sql
SELECT column_name
FROM table_name
WHERE column_name operator (
    SELECT column_name
    FROM table_name
);
```

Сначала выполняется **внутренний запрос**, затем его результат используется во **внешнем запросе**.

---

# 2. Подзапрос в WHERE

Пример: найти студентов, у которых оценка выше средней.

```sql
SELECT name
FROM students
WHERE id IN (
    SELECT student_id
    FROM grades
    WHERE grade > (
        SELECT AVG(grade) FROM grades
    )
);
```

---

# 3. Подзапрос с оператором IN

Оператор `IN` позволяет проверить, входит ли значение в набор данных.

Пример:

```sql
SELECT name
FROM students
WHERE id IN (
    SELECT student_id
    FROM grades
);
```

Запрос выводит студентов, которые имеют хотя бы одну оценку.

---

# 4. Подзапрос с оператором EXISTS

Оператор `EXISTS` проверяет, возвращает ли подзапрос хотя бы одну строку.

```sql
SELECT name
FROM students s
WHERE EXISTS (
    SELECT *
    FROM grades g
    WHERE g.student_id = s.id
);
```

---

# 5. Подзапрос в FROM

Подзапрос можно использовать как временную таблицу.

Пример:

```sql
SELECT course_id, avg_grade
FROM (
    SELECT course_id, AVG(grade) AS avg_grade
    FROM grades
    GROUP BY course_id
) AS course_avg;
```
